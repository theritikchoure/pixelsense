# PixelSense - Email Tracking Application

This application allows you to send tracked emails with a tracking pixel to monitor email opens. It is built using Express and TypeScript, with MongoDB for tracking email opens.

## Technologies Used

- **Node.js**: Server-side JavaScript environment.
- **Express**: Web framework for Node.js.
- **TypeScript**: Superset of JavaScript that adds type safety.
- **MongoDB**: NoSQL database for storing tracking data.
- **Nodemailer**: Library for sending emails.

### Note - 

Make sure to comply with privacy laws and regulations when tracking email opens. Always inform recipients that tracking is in place.
